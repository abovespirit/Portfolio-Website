Simple Sword

During my Modeling 2 class at UCF, I was tasked with creating a short and simple project using Maya. This was one of my first full experiences of following the game industry pipeline and baking the high poly onto the low. The baking process was new to me, but I was determined to learn and understand it. Using Substance Painter, I was able to create realistic and detailed textures that added personality to my model. This project not only allowed me to develop my technical skills, but it also showed me the importance of following a pipeline in order to create industry-standard assets.
